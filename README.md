## Project Overview
This project analyzes Airbnb listings in Albany, New York to uncover pricing patterns,
room type trends, availability behavior, and geographic insights using Python.

## Tools & Technologies
- Python
- Jupyter Notebook
- Pandas, NumPy
- Matplotlib, Seaborn
- Folium (for maps)

## Dataset
Source: Inside Airbnb  
City: Albany, New York  
Listings analyzed: 478

## Key Analysis Performed
1. Data cleaning and preprocessing
2. Price distribution and outlier analysis
3. Room type vs pricing comparison
4. Availability and review behavior
5. Neighborhood-level price analysis
6. Interactive geographic visualization

## Interactive Map
An interactive Folium map is included as an HTML file.
Open `albany_airbnb_price_map.html` to explore listing locations and price clusters.

## Key Insights
- Albany is primarily a budget to mid-range Airbnb market
- Entire homes are significantly more expensive than private rooms
- High-priced listings cluster in specific neighborhoods
- Listings with many reviews tend to have competitive pricing
